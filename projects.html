Project 1 - Colorado Native Plant Website

Project Link -- PLEASE NOTE: I currently am unable to host to any webpage due to issues with vite

This website uses the Three.js library and bundling with vite in order to create an interactive experience in which users can learn more about different types of plants native to Colorado. It includes interative features such as mousing over a collage of different images, orbiting around 3D models, and finding more resources to learn more.
