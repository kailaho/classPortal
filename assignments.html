Assignment 1 - JavaScript and the DOM

  • Assignment Link
  • GitHub Code

For this assignment, we were asked to use JavaScript to interact with the Document Object Model in order to modify one CSS property and one non-CSS property using document queries and event listeners. I chose to create a grid and manipulate the background color (CSS Property) when the user mouses over the "Hover Here" text, and a piece of text that changes when it is clicked (non-CSS Property). From this assignment, I learned how to use document query selectors, and more specifically that you must iterate through each incident of a class member if you use the getElementbyClass selector.
