Assignment 1 - JavaScript and the DOM

  • Assignment Link
  • GitHub Code

For this assignment, we were asked to use JavaScript to interact with the Document Object Model in order to modify one CSS property and one non-CSS property using document queries and event listeners. I chose to create a grid and manipulate the background color (CSS Property) when the user mouses over the "Hover Here" text, and a piece of text that changes when it is clicked (non-CSS Property). From this assignment, I learned how to use document query selectors, and more specifically that you must iterate through each incident of a class member if you use the getElementbyClass selector.

Assignment 2 - HTML Forms

  • Assignment Link
  • GitHub Code

As part of this assignment, we were tasked with creating a form consisting of various types of input elements. I used one button tag for the submit button, a text, color and date type for the different input tags, and a select/option input field. I also used JavaScript event listeners to change the text of a paragraph tag when the submit button is clicked. Although the form is relatively simple, it was a good reminder and practice exercise on the different capabilites and options for HTML forms.

Assignment 3 - JavaScript Debugging

  • Assignment Link
  • GitHub Code

To complete this assignment, we were asked to debug a file containing JavaScript code that gives the user a quiz about the moons of the planets through prompts and alerts, and then display their score out of three once they complete the quiz. The first question should be random, and then iterate through the other two questions. This assignment was a good reminder of many common mistakes that can be made when coding JavaScript, and one that was a good reminder for me especially was using the correct query selector when selecting elements from a document.

Assignment 4 - APIs and JSON

  • Assignment Link
  • GitHub Code

This assignment uses the coolor.io API in order to generate a random color palette that is then applied to the gnome graphic to the right. In order to replace the colors, I iterated through each pixel of the image, cheked if it match the old color, and then replaced it with the new color generated from the color palette. The API outputs five colors, which change the gnome's hat, shirt, mittens, pompom, and shoes individually. This was a very challening assignment for me, but I learned a lot about APIs, using "canvas" in HTML, for loops, and got lots of practice debugging using console.log and developer tools.
